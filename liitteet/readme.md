# Liite 1: Signaaleja

## Signaaleja taloudellis-organisatorisen kompositiomme muuttumisesta ohjelmoitavaksi

<sup>ECSA Labs</sup>

| USA rakentaa strategista Bitcoin-reserviä                                          | → Kansallisvaltiot alkavat käyttää ohjelmoituja rahoitusprotokollia strategisina varantoina. Bitcoin ei ole enää vain omaisuus, vaan ohjelmoitava rahajärjestelmä – sen koodin valinta on geopoliittinen päätös. |
| ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| BRICS-maat kehittävät lohkoketjupohjaista reservivaluuttaa haastamaan dollaria     | → Globaali valuuttajärjestelmä siirtyy protokollien tasolle. Reservivaluutta ei ole enää kansallisen keskuspankin poliittinen mandaatti, vaan lohkoketjuverkoston ohjelmoitu funktio.                            |
| Keskuspankit suunnittelevat digivaluuttoja                                         | ​​→ Rahan liikkumisen ehdot voidaan ohjelmallisesti määritellä: kuka saa käyttää, mihin, milloin. Raha ei ole vain vaihdannan väline, vaan käytettävyyden ohjelmointirajapinta.                                  |
| WeChat yhdistää maksamisen, viestinnän, identiteetin – ja valvonnan                | → Sosiaalinen ja taloudellinen toiminta on integroitu ohjelmallisesti hallittavaksi järjestelmäksi. Käyttöliittymä = järjestelmä = hallinnan muoto.                                                              |
| Joukkorahoitusalustat kuten Patreon ja Kickstarter ovat protokollia                | → Taloudellinen organisoituminen toteutuu ennalta määritellyissä säännöissä, joissa käyttäjä voi osallistua, mutta ei määritellä itse sääntöjä. Ne ovat protokollia, eivät enää pelkkiä alustoja.                |
| SoFi ja muut p2p-lainaprotokollat arvioivat sosiaalista luottamusta algoritmisesti | → Luottoluokitus ei enää vaadi instituutiota – se on dataohjattua logiikkaa, joka on ohjelmoitavissa. Algoritmit määrittävät kenen luottamus “kannattaa” ja millä ehdoin.                                        |
| Keikkatyöläiset työskentelevät protokollille joita ei valvo kukaan ja kaikki       | → Organisaatiot eivät ole enää yrityksiä, vaan ohjelmoituja sopimuslogiikoita, joiden osallisuus ja hallinta ovat koodattuja.                                                                                    |
| MakerDAO tarjoaa hajautetun vakuusjärjestelmän                                     | → Rahoitusinstrumentit (kuten vakuudet, lainananto) toimivat itsenäisinä ohjelmoituina järjestelminä, eivätkä perustu pankki-instituutioon.                                                                      |
| Tokenit tekevät käyttäjistä omistajia, hallinnoijia ja operoijia                   | → Arvon ja hallinnan jakautuminen ei ole enää johdettavissa oikeudesta tai sopimuksista – vaan koodatuista osallisuuden logiikoista.                                                                             |
| OpenSea ja NFT:t luovat talouksia niukkuudesta                                     | → Arvo on määritelty protokollan kautta: ohjelmoitu niukkuus, siirrettävyys ja omistus. Talous on ohjelmoitavaa merkityksenmuodostusta.                                                                          |
| DAO-työkalut mahdollistavat talousverkoston ilman yhtiömuotoa                      | → Organisoituminen ei edellytä lakia, vaan protokollaa. Yhtiömuoto on koodia                                                                                                                                     |
| Wall Street Bets järisytti markkinoita meemien avulla                              | → Koordinointi, spekulaatio ja toimijuus tapahtuivat ohjelmoidun viestinnän (Redditin) sisällä. Taloudellinen voima = keskitetty huomio + hajautettu toiminta.                                                   |
| Meme-tokenit rahastavat kollektiivista huomiota ja spekulaatiota                   | → Talous toimii kulttuurisena ohjelmointina: arvo syntyy memetiikasta ja jaetusta affektista, joka tokenoidaan.                                                                                                  |
